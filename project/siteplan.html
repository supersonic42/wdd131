Website plan

Site Name

Pixel Pulse - because my website would be about Video Games and the tiniest part of a game is a pixel. Its changes and game pulses.

Site Purpose

My website will show video games news, various articles, guides tips and tricks. Anyone could find their favourite game on this website and get a lot of information, helping him to success in game. Also there will be some materials like Game Mods available to download.

Scenarios

  • Speedrun guide for Dark Souls.
  • Where to find Mankrik's wife doing the quest in World of Warcraft?
  • When new expansion for Elden Ring will be released?

Color Scheme

Retro Pixel Palette

  • Primary: Bright Red (#FF4D4D)
  • Secondary: Sunshine Yellow (#FFD700)
  • Background: Black (#000000)
  • Accent: Cyan (#00FFFF)
  • Text: White (#FFFFFF)

Typography

Roboto - is my main font for everything on the website.
